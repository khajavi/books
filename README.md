# Books

_Books_ is a Gtk+ 3.0 application to view and manage books in EPUB format.
Translation are managed over at [Transifex][]. Also Travis sez
[![Build Status](https://travis-ci.org/matze/books.png)](https://travis-ci.org/matze/books).

[Transifex]: https://www.transifex.com/projects/p/books/

## Installation

Configure, compile and install _Books_ with

    ./autogen.sh
    ./configure
    make && sudo make install

## Dependencies

* Gtk+ 3.0
* GtkWebKit
* libarchive
* sqlite3
