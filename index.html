Books

Books is a simple Gtk+ 3.0 application to view and manage books in EPUB format.

Installation

Configure, compile and install Books with

./autogen.sh
./configure
make && sudo make install

Travis says .

Dependencies

  • Gtk+ 3.0
  • WebKitGtk
  • libarchive
  • sqlite3

If you want to see Books in your language, don't hesitate to contribute by translating Books at Transifex.
